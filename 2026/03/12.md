 ##  わかったこと
### TypeScript

- コンストラクタの省略記法
    ```ts
    // フル記述
    class User {
      name: string; // 1. 宣言
      constructor(name: string) { // 2. 引数
        this.name = name; // 3. 代入
      }
    }
    

    // 省略形
    // 自動でプロパティ宣言と代入が行われる
    class User {
      constructor(public name: string) {}
    }
    ```
