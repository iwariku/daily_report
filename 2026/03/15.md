##  わかったこと
### TypeScript
 - `typeof`・・・値から型を取得。既存のオブジェクト変数などから、その構造(型定義)を抽出する。
```
const appConfig = {
  title: 'MyApp',
  version: '1.0.0',
  theme: 'dark',
};

type Config = typeof appConfig;
```
と記述すると
```
type Config = {
    title: string;
    version: string;
    theme: string;
}
```
という型が生成される。

- `keyof`・・・型からキーを取得。型のプロパティ一覧を`Union`型("a" | "b"...)として取得する。
```
const appConfig = {
  title: 'MyApp',
  version: '1.0.0',
  theme: 'dark',
};

type Config = typeof appConfig;

type ConfigKeys = keyof Config;
```
と記述すると
```
type ConfigKeys = "title" | "version" | "theme"
```
というプロパティのUnion型が生成される。
