 ##  わかったこと

- `select`はチャネル通信(送受信)ができるものを選んで実行する。

- `doneチャネル`は並行処理でキャンセルを伝えるための共通ルール(慣習)

- `sync.WaitGroup`は「複数のgoroutine」が全員終わるまで待つための仕組み。

- `interface{}やany`の中身をチェックするために、「型スイッチ(Type Switch)」という構文がある。
    ```
    // 型スイッチの例
    switch concreteValue := v.(type) {
    case int:
        // vがintだった時の処理（concreteValueはint型として扱える）
    case string:
        // vがstringだった時の処理（concreteValueはstring型として扱える）
    }
    ```
