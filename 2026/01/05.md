- 下記のように`switch`の後に条件分岐を記載せずに、`case`の部分に記載する記述方法もある。
    ```
	t := time.Now()
	fmt.Println(t.Hour())
	switch {
	case t.Hour() < 12:
		fmt.Println("morning")
	case t.Hour() < 23:
		fmt.Println("afternoon")
	}
    ```
