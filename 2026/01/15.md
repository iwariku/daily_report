 ##  わかったこと

- `fmt.Println()`などには戻り値があること。

- `fmt.Println`の引数の値が`String() string`を持っていると、その`String()`を呼び出さなくても、`String()の内容が呼び出される。`

    ```
    func (m MyIntSlice) String() string {
    	strValues := make([]string, 0, len(m))
    	for _, v := range m {
    		strValues = append(strValues, strconv.Itoa(v))
    	}
    	return "[" + strings.Join(strValues, ", ") + "]"
    }

    func main() {
    	m := MyIntSlice{1, 2, 2, 3, 3, 3, 4, 5}
    	fmt.Println(m.Unique())

    // fmt.Println((m.Unique()))と呼ばれているが、String()を呼び出すので、出力結果に「,」がつく
    }
    ```
