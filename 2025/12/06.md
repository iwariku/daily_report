 ##  わかったこと

- `const arry2 = [...arry1]`のように`[...配列]`としている場合、`[...arry1]`と同じ配列を新しく作成し、arry2に代入する

- `FPS(Flames Per Second)`とは、1秒間当たりの画面(フレーム)更新頻度

- 非同期処理は一時的にメインスレッドから処理が切り離される

- タスクキューとは、実行待ちの非同期処理の行列<br>非同期処理の実行順を管理
- `FIFO(First In, First Out)`キューの仕組みを「先入れ先出し」という

- `Promise`を使った構文では、`resolve()`または、`reject()`が呼ばれるまでは、`.then`や`.catch`などは実行されないので、この特徴を使い、`Promise`のコールバック関数に非同期処理を組み込むことができる

- 非同期の処理を繋げるためには、`then`メソッドの中のコールバック関数の`return`に`Promise`のインスタンスを設定する

- `fetch関数`とは、サーバー上から取得できる関数。返却値は`Promise`となる

- `モジュール`とは、ソースコードを機能毎に分割して、メンテナンスしやすくする仕組み

- `Proxy`とは、プロパティの操作に独自の処理を追加するためのオブジェクト

- `JSON.parse`<br>JSON -> Object
- `JSON.stringify`<br>Object -> JSON

- `Storage`とは、ブラウザの保存領域にデータを格納するためのオブジェクト<br>`localStorage`を使う
