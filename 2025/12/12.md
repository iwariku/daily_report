 ##  わかったこと

- コンポーネントの名前の最初は大文字にする

- 関数のコンポーネントとして返却していくタグは、何か1つのタグで囲わなくてならない。
    ```
    // これはOK
      return (
        <div>
          <h1>こんにちは</h1>
          <p>こんばんは</p>
        </div>
      );
   
    <div>をわざわざ書かない方法として<></>で記述できる。これをフラグメントという

      return (
       // これはNO<h1>と<p>が同じ階層にいるから
        <h1>こんにちは</h1>
        <p>こんばんは</p>
      );
    ```

- Reactのコンポーネントファイルには拡張子`.jsx`をつける

- HTMLのタグの中で扱うイベントや、CSSはキャメルケースで書く。
    `<button onClick=>`

- 関数コンポーネントのreturnしてある以降のHTMLを記述する部分で`{}`を書くと、その部分はJavaScriptとして認識される
    `<button onClick={JavaScriptとして認識される部分}>`
