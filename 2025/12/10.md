 ##  わかったこと

- 三項演算子<br>ある条件 ? 条件がtrueの時 : 条件がfalseの時<br>`const val = 1 > 0 ? "trueです" : "falseです"`
 
- `||` は左側がtruthyの時、その時点で返却する。以下の場合だと、num = 100でtruthyであるので、`||`の左側で返却される。
    ```
    const num = 100;
    const fee = num || '金額未設定です';
    ```

- `&&`は両方ともtruthyの時、右側の値を返却する。以下の場合だと、どちらもtrueなので、`&&`の右側の`何が設定されました`が返却される。
    ```
    const num2 = 200;
    const fee2 = num2 && '何か設定されました;
    ```
