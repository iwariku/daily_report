 ##  わかったこと

- `while文`はループの中で条件式に関わる変数を自分で更新しないと、無限ループが起こる。
- `"use strict"`とJSファイルの先頭に記述することで、厳格モードにすることができる。<br>(例)変数宣言時に`let`や`const`をつけ忘れたときにエラーを出してくれる。<br>これは変数宣言時に`let`や`const`をつけなくても問題なくプログラムが動くというのを、つけないといけないよという厳格チェックしてくれる。
- プロパティ<br>データが持っている情報<br>`()`はつけない
- メソッド<br>データに対して使える動作や命令<br>`()`をつけて実行する。
- `colors.forEach((color, index) => {})`<br>colorsの　要素が、color,とindexに入り、アロー関数のcolorsの要素の分だけ処理が行われる。
- `mapメソッド`<br>配列のすべの要素に同じ処理を加えて、その結果を新しい配列として作り出すメソッド。<br>元の配列は変更せずに、加工した新しい配列が欲しいときんい使う
- `filterメソッド`<br>配列のすべての要素をチェックして、条件にあうものだけを生き残らせて、新しい配列を作り出すメソッド
- constで定義してあるオブジェクトを編集できるのは、const a = {11, 22}の変数aはオブジェクトのアドレス持っているので、constの直接値を編集するとエラーが発生するというのには該当しないから
- デフォルト引数<br>関数の呼び出し時に、指定の引数が呼び出されなかった場合に、デフォルトの値を出力するように設定することができる。<br>`function 関数名() {sample = "デフォルト値"}`でデフォルト値を設定することができる
- 戻り値を返す関数で`return`を記述しないと処理の結果ではなく`undefined`が返される
- メソッドとは、オブジェクトが持つ関数の事
- `this`は自オブクジェクトという意味。<br>`this.name`はこのオブジェクトのnameという変数という意味
- メソッドを呼び出す際は`オブジェクト名.メソッド名()`と記述する
- ラッパーオブジェクト<br>普段、文字列や数値はメモリ節約のためにシンプルな「値」として存在している。<br>その値に対して`.length`や`.toUpperCase()`などの操作をするとJavaScriptは一時的にオブジェクトを提供する。
- アロー関数とthisの継承の仕組み
```js
const object = {
  name: "日本",
  region: "アジア",

  showInfo: function () {
    //functionキーワードはthisを持つ
    console.log(this.name); // 日本

    const getResion = () => {
      return this.region; //アジア
    };
  },
};

```
    fucntionキーワードで定義されたメソッドは自分自身のthisを持つ
    アロー関数は自分自身のthisを持たない
    アロー関数は生まれた場所(レキシカルスコープ)のthisを引き継ぐ
