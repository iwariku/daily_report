 ##  わかったこと

- 関数の引数にオブジェクトそのものを渡す場合は`(オブジェクト名)`と記述し、代入しながら受け取る場合は、`({ name, age })`と記述する。
- 高階関数<br>「引数として関数を受け取る」または「戻り値として関数を返す」関数のこと
- クロージャ<br>関数と、その関数が生まれた場所をセットで持ち歩く仕組み<br>記憶される変数は外部からの変更は受け付けないため、オブジェクトのカプセル化と同じ挙動を関数を使って書いている
```js
function 外部関数() {
  const 記憶される変数 = 値;
  return function 内部関数() {
    // 内部関数は外部関数のスコープの変数を記憶している。
    記憶される変数を使った処理;
  };
}
```
