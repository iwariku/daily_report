 ##  わかったこと

- オブジェクトのメソッドで`this`を使用する場合は、`function ()`で記述し、`this`を使わない場合はアロー関数で記述する。
- 分割代入<br>配列の要素を変数に格納する<br>teamという配列の要素を`captain`,`member1`,`member2`に格納する
    ```js
    // 配列の分割代入
    const team = ["Taro", "Jiro", "Saburo"];
    const [captain, member1, member2] = team;
    ```
    `...members`とすることで`team`配列の1つ目の要素は`captain`に格納、他の要素は新しく作った配列`members`に格納する
    ```js
    // 配列の分割代入
    const [captain, ...members] = team;
    console.log(captain, members);   
     ```
    配列同様に`const {分割代入する変数1,分割代入する変数2} = オブジェクト`
    ```js
    // オブジェクトの分割代入
    const userProfile = {
      userName: "Taro",
      userAge: 25, 
    }
    const { userName, userAge } = userProfile;
     ```
- スプレッド構文<br>`const copyArray = [...コピー元配列]`と記述することで、コピー元配列と同じ要素を持った配列を作成することができる

- カプセル化<br>オブジェクト内のプロパティに対して、内部参照のみできるようにする<br>カプセル化したいkeyやメソッドの前に`#`をつけることでカプセル化することができる
